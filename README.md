# FreeAI

> A collection of reusable AI code snippets, tools, and integrations maintained by WSU-EIT.

FreeAI demonstrates practical patterns for working with AI across a variety of contexts — from chatting with LLMs to accessibility checking, Blazor UI extensions, smart sheets, plugin systems, and more. Each project area is self-contained and documented following the [FreeAI Documentation Standard](./DOCS_STANDARD.md).

---

## 📁 Project Areas

| Folder | Description |
|--------|-------------|
| [`ChatWithAI`](./ChatWithAI) | Snippets and examples for interacting with AI chat models |
| [`FreeA11yChecker`](./FreeA11yChecker) | AI-assisted accessibility checking tools |
| [`FreeBlazorExtended`](./FreeBlazorExtended) | Blazor UI components and extensions enhanced with AI capabilities |
| [`FreeLLM`](./FreeLLM) | Low-level LLM integration patterns and utilities |
| [`FreeGLBA`](./FreeGLBA) | AI patterns for GLBA-related workflows |
| [`FreeManager`](./FreeManager) | Management tooling and orchestration with AI |
| [`FreePlugins`](./FreePlugins) | Plugin architecture for AI-powered extensibility |
| [`FreeServices`](./FreeServices) | Background service patterns integrating AI |
| [`FreeServicesHub`](./FreeServicesHub) | Hub for aggregating and routing AI services |
| [`FreeSmartsheets`](./FreeSmartsheets) | AI-driven Smartsheet automation and integration |
| [`FreeTools`](./FreeTools) | Developer tooling: screenshot pipelines, workspace reports, and more |
